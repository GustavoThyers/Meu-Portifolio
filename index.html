Projeto desenhos Flix

Este projeto de site de desenhos animados foi desenvolvido usando HTML, CSS e PHP puro e possui como objetivo principal fornecer uma plataforma de entretenimento para os usuários, onde eles possam assistir filmes e séries animadas.

A plataforma permite a categorização dos desenhos, além de possuir recursos de login e registro para os usuários.
A motivação por trás do projeto foi minha necessidade pessoal em encontrar um site para assistir desenhos animados. A partir dessa necessidade, o projeto foi desenvolvido para fornecer uma solução para outras pessoas com interesses semelhantes.

O projeto foi desenvolvido inteiramente por mim, que enfrentei alguns desafios ao criar os relacionamentos do banco de dados e do front end, pois não é muito minha area. No final, o resultado foi um site funcional e intuitivo, que atende às necessidades de entretenimento dos usuários.

Além de oferecer entretenimento e categorização de filmes e séries animadas, o projeto de site de desenhos que desenvolvi também conta com páginas administrativas protegidas e exclusivas para os administradores do sistema. Com essas ferramentas, os administradores têm o controle total sobre o site, podendo adicionar novos desenhos e episódios, visualizar informações sobre os usuários logados e alterar as configurações do sistema. Tudo isso é feito com segurança, garantindo que apenas os administradores tenham acesso a essas funcionalidades.

DevForum

Com o objetivo de criar um ambiente de discussão online, desenvolvi um projeto pessoal de um fórum utilizando HTML, CSS e PHP.
O fórum consiste em uma plataforma onde os usuários podem fazer perguntas e outros usuários podem responder, semelhante a um sistema de comentários.

É possível visualizar o nome ou apelido de quem fez a pergunta e a busca é realizada através do título da pergunta. O projeto foi desenvolvido com foco no back end, por isso o front end ficou um pouco a desejar. Mesmo assim, o sistema de login e registro foi implementado com sucesso. Trabalhei sozinho no projeto e em apenas três dias consegui finalizá-lo.

Apesar de não ter enfrentado grandes desafios durante o desenvolvimento do fórum, o projeto foi desafiador para mim, pois tive que garantir que todas as perguntas estivessem relacionadas a suas respectivas respostas e que o sistema de busca funcionasse adequadamente. No final, estou satisfeito com o resultado final e espero que os usuários possam se beneficiar dessa plataforma de discussão online.
